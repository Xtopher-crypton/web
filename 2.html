Questão 2 - Estrutura e Sintaxe do CSS

O que é uma regra CSS

Uma regra CSS é composta por um seletor, propriedades e valores. O seletor indica qual elemento será estilizado, as propriedades definem o que será mudado, e os valores definem como será mudado.

Exemplo de regra:

button{
    padding: 10px 20px;
    background-color: blue;
    color: white;
}

Neste exemplo:

  • Seletor: button
  • Propriedade: padding, background-color, color
  • Valor: 10px 20px, blue, white

Pseudo-classe

Uma pseudo-classe define um estado especial de um elemento. Exemplo: :hover quando o mouse passa sobre o elemento.

Este texto tem pseudo-elementos ::before e ::after

Animação com @keyframes

@keyframes permite criar animações. Você define os estados em 0%, 50%, 100% etc.
