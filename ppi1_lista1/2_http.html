Protocolo HTTP

O HTTP (HyperText Transfer Protocol) é o protocolo utilizado para comunicação na Web. Ele permite que um navegador envie requisições para um servidor e receba respostas.

Funcionamento do HTTP

Quando você acessa um site, o navegador envia uma requisição ao servidor. O servidor processa essa requisição e retorna uma resposta.

Fluxo de requisição e resposta.

Modelo cliente-servidor.

Outra representação da comunicação HTTP.

Exemplo de Requisição

GET /index.html HTTP/1.1
Host: www.exemplo.com
User-Agent: Mozilla/5.0

Campos da Requisição

  • Método: GET, POST
  • URL: endereço solicitado
  • Headers: informações adicionais

Exemplo de Resposta

HTTP/1.1 200 OK
Content-Type: text/html

Campos da Resposta

  1. Status: 200 OK, 404 Not Found
  2. Headers: tipo de conteúdo
  3. Body: conteúdo retornado

Resumo

O HTTP é essencial para o funcionamento da Web, permitindo a comunicação entre cliente e servidor.
