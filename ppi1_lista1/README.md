# Projeto PPI1 - Lista 1
Este projeto foi desenvolvido para a disciplina de Programação para Internet.

## Descrição
O objetivo deste trabalho é aplicar os conceitos básicos de HTML e CSS aprendidos em aula, incluindo estrutura de páginas, uso de tabelas, formulários e estilização.

## Conteúdo
O projeto contém os seguintes arquivos:

- 1.html: Estrutura básica de HTML e explicação sobre a diferença entre Internet e Web  ;
- 2_http.html: Explicação do protocolo HTTP com exemplos de requisição e resposta  ;
- 3_tabela_uf.html: Tabela de estados organizada por região  ;
- 4_curriculo.html: Página de currículo com informações pessoais  ;
- 4_contato.html: Formulário de contato com validação básica  ;
- style.css: Arquivo de estilos aplicado às páginas .
