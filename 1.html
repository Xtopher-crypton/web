O que é HTML

HTML é uma linguagem de marcação usada para criar páginas web. Ela usa tags para estruturar o conteúdo, como títulos, parágrafos, listas e links. O navegador lê essas tags e exibe o conteúdo formatado.

Qual o papel dele na Web

O HTML é a base de toda página web. Ele define a estrutura e o conteúdo que será exibido. Sem HTML não teríamos páginas web. O CSS é usado para estilizar e o JavaScript para adicionar interatividade, mas o HTML é sempre a base.

Exemplo de lista

  • CSS estiliza a página
  • JavaScript respode açoes
