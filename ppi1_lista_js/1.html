O que é JavaScript?

O JavaScript foi criado para tornar as páginas web mais interativas. Inicialmente, era executado apenas no navegador para manipular o DOM e responder a eventos do usuário. Com o tempo, o Node.js permitiu sua execução no servidor, ampliando suas aplicações para desenvolvimento backend. No navegador, o JavaScript opera em um ambiente "sandbox", que restringe seu acesso ao sistema operacional para garantir segurança. Objetos globais como `document` (para interagir com o HTML), `console` (para depuração) e `alert` (para exibir mensagens) são fundamentais para o desenvolvimento web.
