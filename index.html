Commandos Git

  • git init: cria um repositorio
  • git add: Adiciona um arquivo no stage
  • git status: exibe o estado do repositorio
  • git commit -m "msg": cira um commit
  • git log: exibe o commits do repositorio, da branch atual
  • git checkout referencia: Recupera o conteudo da referencia para MK
  • git log: ver os commit feitos
