Commandos Git

  • git init: cria um repositorio
  • git add: Adiciona um arquivo no stage
  • git status: exibe o estado do repositorio
  • git commit -m "msg": cira um commit
  • git log: exibe o commits do repositorio, da branch atual
  • git checkout referencia: Recupera o conteudo da referencia para MK
  • git log --oneline: ver os commit feitos
  • git remote add nome url: definir um repositorio remoto
  • git push url branch: envia a branch para o repositorio remorto
  • git checkout tag (master): se trata de um ponto de conteudo dos commit
  • git log --oneline: Ver os commit de maneira mais simples

  • git checkout -b novabranch
